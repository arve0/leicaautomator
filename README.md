## Overview

Automate scans on Leica SPX microscopes

## Requirements

* Python (2.7, 3.4)

Install using `pip`...


To build the documentation, you'll need sphinx:

```bash
pip install -r docs/requirements.txt
```

To build the documentation:

```bash
make docs
```
